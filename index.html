Projects


Motion Control MIDI Interface

I helped create a motion control interface for music-making applications at a hackathon in California. We used a Leap Motion controller, as well as a prototyping touch interface provided by Synaptics. The application tracked the movement of your hand, where each movement axis, as well as rotation, and the tilt of each finger, could be paired to a different parameter in any MIDI compliant application.

See the project and source code on Devpost
Disk Sorting Machine

I built a machine to sort black and white disks on some electronics hobbyist hardware, and the TU/e's properietary processor, the PP2. The machine ran on roughly 600 lines of assembly code, and works by detecting different levels of light reflected by the different disks. The machine features robust error detection and user feedback through a seven segment display.

See the source code on Github
